Template for Multi-Layered
Language Learning Resources*

*Currently under construction.

Introduction

This is an open-source template for multi-layered language learning resources template intended to address the needs of many kinds of learners. It consists of nine components. All components are optional.

  1. About the Materials
    • A description about the language, the speakers, and the members involved in the creation of these resources.
  2. Links to Mobile Apps
    • A brief description of the mobile apps, and links that direct users to the download page.
  3. Sounds and Letters
    • A grid of letters including soundfiles for each letter.
  4. Everyday Expressions
    • This page contains common phrases and sentences.
  5. Talking Dictionary
    • A dictionary including soundfiles for each entry.
  6. Phrasicon
    • A bank of phrases and sentences including soundfiles.
  7. Basic Sentence Structures
    • A brief grammatical sketch of the language.
  8. Stories and Text
    • Stories and texts of the language.
  9. Contact
    • A page with contact information for users to get in touch with the creators of the resources..

Making Changes to the Template

The template was created with non-programmers in mind. That being said, applying changes will require some level of tech-savviness. This documentation is written in attempts to guide potential users through the process. The template uses the following computer languages: HTML, CSS, JavaScript, XML, and PhP.

HTML (HyperText Markup Language) sets the structure of the web page and displays the content. It is known as a markup language because the code is made up of tags enclosed in angle brackets. For example, <body> is a 'body' tag. Comments can be added to HTML, and are not processed by the computer - they are only meant to be read by humans. Comments in HTML look like this: <-- This is a comment. -->. They will used to indicate where changes should be made. Note that making changes elsewhere may have drastic changes when viewing the web page.

CSS (Cascading Style Sheets) describes the layout, color and font of the web page. For example, italicized, bolded, or colored font are handled by CSS. JavaScript is a programming language that deals with the interaction between the user and the computer. The Javascript code found in this template should be not be altered. XML is a format where data is stored. This will be discussed later on. Finally, PhP is in charge of retrieving data from the XML formats so that they may be displayed on the web page. Comments in PhP look like this: /* This is a comment. */.
